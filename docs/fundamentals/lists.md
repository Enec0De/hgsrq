列表表格 { #list-and-tab }
==========================

-   约定有序列表仅用符号 `1.` 进行排列。
-   普通列表和任务列表，不使用超过两层的嵌套[^1]。

[^1]: 过多层的嵌套容易导致无意义的复杂化，建议拆分。

定义列表 { #definition }
------------------------

``` markdown title="Definition list"
`佐恩引理`

:   在任何一非空的偏序集中，若任何链（即全序的子集）都有上界，则此偏序集内必然存在（至少一枚）极大元。
```
